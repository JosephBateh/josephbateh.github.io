Latest Project

Minecraft Server

A Minecraft Server running on a Raspberry Pi 2

New

This was a weekend project of mine. I had an original Raspberry Pi running a Minecraft server, but the performance was less than ideal as the original Pi only had one processing core and 512MB of RAM. A co-worker was gracious enough to give me his Raspberry Pi 2 to host the server. With four processing cores and 1GB of RAM the performance has been signficantly improved. Feel free to build some stuff on it! It is running vanilla Minecraft with the world edit mod.
